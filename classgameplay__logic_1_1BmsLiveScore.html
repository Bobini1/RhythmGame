Detailed Description

The score that gets updated during gameplay.

The BmsLiveScore class keeps track of the current score, combo, gauge and judgement counts during gameplay. It provides methods to add hits and retrieve the current score state.

uint64_t gameplay_logic::BmsLiveScore::randomSeed

The random seed used for note order shuffling.

This seed is used to initialize the mt19937 random number generator for shuffling notes according to the selected note order algorithm.

Note
