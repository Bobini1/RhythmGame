◆ languages

QStringList resource_managers::Languages::languages
read

The list of available languages.

This is a list of language codes, e.g. "en", "de", "fr", etc. The list is populated from the available translation files, defined in theme.json of each theme.

◆ selectedLanguage

QString resource_managers::Languages::selectedLanguage
read

The currently selected language.

This is the language code of the currently selected language. It will change the language for the whole application.

Note
You can't set this property from QML, set GeneralVars::language on the resource_managers::ProfileList::mainProfile instead.
