Generate a permutation of notes.

The permutation is generated using the given algorithm and seed.

Warning
For S-Random, this function relies on total stability of note generation. If we ever change how we load notes, replays WILL break.
Parameters
notesThe notes to shuffle.
algorithmThe algorithm to use.
seedThe seed to use.
Returns
The result of the shuffle, including the new seed and the new column order.
