auto support::generatePermutation ( std::span< std::vector< charts::BmsNotesData::Note > > &  notes,
resource_managers::NoteOrderAlgorithm  algorithm,
uint64_t  seed,
bool  k5,
bool  usePre130 
) -> ShuffleResult

Generate a permutation of notes.

The permutation is generated using the given algorithm and seed.

Warning
For S-Random, this function relies on total stability of note generation. If we ever change how we load notes, replays WILL break.
Parameters
notesThe notes to shuffle.
algorithmThe algorithm to use.
seedThe seed to use.
k5Whether to treat the notes 5k as 7k (legacy behavior)
usePre130Whether to use the broken pre-1.3.0 version of the s-random shuffle algorithm (legacy behavior)
Returns
The result of the shuffle, including the new seed and the new column order.
