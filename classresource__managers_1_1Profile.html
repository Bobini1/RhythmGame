Creates a profile object living in the given database.

If the profile doesn't exist, it will be created.

Parameters
mainDbPathPath to the main song database file. Has to exist.
dbPathPath to the profile's database file. Doesn't have to exist.
themeFamiliesThe available theme families.
assetsPathsPaths to the avatar folders.
networkManagerThe network manager to use for online operations.
parentQObject parent.

Download scores from the server that are missing locally.
