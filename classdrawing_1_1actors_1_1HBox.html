A HBox is a parent that arranges its children horizontally.

When width is managed, children with managed width are stretched equally to fill space.

Base classes

class AbstractBox
AbstractBox is an abstract base class for VBox and HBox. It provides the concept of a size mode, which determines how the size of the box is calculated.
