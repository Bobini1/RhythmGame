Detailed Description

Tracks the progress of an asynchronous beatoraja replay import.

Instances are created by BeatorajaReplayImporter::importFolder(). Each processed file (whether imported, skipped, or failed) increments done toward total. The error() signal fires for each failure without halting the operation.
