◆ clearType

QString gameplay_logic::BmsResult::clearType
read

The clear type achieved in the score.

This property indicates the type of clear achieved in the score, such as "HARD", "EXHARD", "FC", "FAILED", etc. For charts played with course gauges, it will always be "NOPLAY".

◆ randomSeed

uint64_t gameplay_logic::BmsResult::randomSeed
read

The random seed used for note order shuffling.

This seed is used to initialize the mt19937 random number generator for shuffling notes according to the selected note order algorithm.

Note
This seed is not used for #RANDOM parsing. Use randomSequence for that.

◆ randomSequence

QList< qint64 > gameplay_logic::BmsResult::randomSequence
read

The random sequence used for parsing the chart.

If the chart uses #RANDOM, this property provides the sequence of random values used to determine the note order. If the chart does not use randomization, this list will be empty.
