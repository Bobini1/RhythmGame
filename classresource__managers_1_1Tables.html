Detailed Description

Manages the download and storage of tables from given URLs.

The Tables class handles downloading table data from specified URLs, storing it in a local cache file, and parsing the data for use in QML. It provides methods to add, remove, reload, and reorder tables, as well as search for specific entries by their MD5 hash.
