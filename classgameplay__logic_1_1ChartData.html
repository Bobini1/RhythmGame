Detailed Description

Metadata and basic stats of a chart.

When loading a chart, the ChartData might be different than what was cached in the database, because the chart file might have changed or it contained #RANDOM, which generates a different chart each time the chart is parsed. In the latter case you can use the randomSequence property to recreate the same chart.
