drawing::actors::Frame class
#include <Frame.h>

A frame that forces the child to render within its bounds.

Contents

  • Reference
    • Base classes
    • Public static functions
    • Public functions
    • Private functions

This is done by passing a separate render target to the child. That render target is then drawn on the parent's render target. If the child is completely outside the parent's bounds, it is not drawn at all.
