drawing::actors::Text class
#include <Text.h>

Text is a leaf actor that displays a string of text.

Contents

  • Reference
    • Base classes
    • Public static variables

Text holds a non-owning pointer to its font.

The best method to change the size of text is setCharacterSize(). setWidth() and setHeight() will stretch the text.
